* Setup
** Download repo
#+begin_src sh
git clone git@github.com:LukinEgor/doom.d.git ~/Projects/doom.d
#+end_src

#+begin_src sh
ln -s ~/Projects/doom.d ~/.doom.d
#+end_src

* Usage
** After some changes in doom.org file you need:
- exec `M-x org-babel-tangle` in doom.org
- exec `M-x doom/reload`
